## 1. 프로젝트 개요

### 1.1 목적
키움증권 HTS 조건식(60/200일 이동평균 돌파)이 포착하는 종목의 성과를 체계적으로
검증하고, 과거 데이터를 기반으로 종목의 "폭발 잠재력"을 사전에 예측하는
스코어링 모델을 개발한다.

### 1.2 핵심 가설
> 60일·200일 이동평균선의 동시 돌파 + 볼린저밴드 극압축 + 거래량 폭발이
> 동시에 발생하는 순간은, 장기 횡보 에너지가 한꺼번에 분출되는 "스프링 해제점"이며,
> 이 시점을 포착하면 100~400%+ 상승의 최적 진입점을 확보할 수 있다.

### 1.3 핵심 성과 (2026-05-20 기준)
- **2년간 백테스트 완료** (2024/05/19 ~ 2026/05/16, 506 영업일)
- **총 3,860건 포착**, 평균 일 7.6건
- **전체 승률 99.5%** (ret_max > 0 기준)
- **2축 스코어링 모델 v2.0** 개발 완료
  - A1등급(최우선): avg_max +23.3%, avg_1m +2.9%, 100%+ 비율 2.7%
  - D등급(패스):     avg_max +14.2%, avg_1m -1.7%, 100%+ 비율 0.0%
  - **A1-D 변별력: avg_max +9.1%p, avg_1m +4.6%p**
- **과적합 검증 통과**: 하락장(P1)에서도 승률 99.3%, avg_max +15.91%

### 2.2 공통 패턴 (6대 DNA)
1. **≥3개월 횡보 압축** (에너지 축적기)
2. **5/20/60/200 이평선 극한 수렴** (갭 <3%)
3. **볼린저밴드 폭 바닥** (BW <20)
4. **포착 전 극저거래량 → 포착일 거래량 폭발**
5. **SuperTrend Red→Green 전환**
6. **RSI 50 부근 중립** (과매수/과매도 아닌 중립대)

→ 위 6가지가 동시에 성립할 때 60일·200일 이평선 동시 돌파가 발생하며,
  이것이 100~400%+ 상승의 트리거가 된다.

## 5. 스코어링 모델 v2.0

### 5.1 모델 구조: 2축 시스템

기존 v1.0(단일 점수)에서 avg_max가 등급과 역전(-3.27%p)되는 문제가 발생.
원인: "안정성"과 "폭발력"이 상충하는 구조 (안정한 종목은 max가 낮고,
폭발적인 종목은 1m이 낮음). 이를 해결하기 위해 2축으로 분리.

**S-Score (Stability, 안정성 0-100점)** — 1개월 실현 수익 예측
**E-Score (Explosion, 폭발력 0-100점)** — 최고 수익 폭발력 예측

| F2. 60/200 이격률 | 20 | <3% (극한수렴) | 하락장 방어력 최강 |

## 6. 프록시 제외 필터

DB 플래그(관리종목, 거래정지 등)가 업데이트되지 않아
일봉 패턴만으로 비정상 종목을 필터링하는 프록시 필터를 구현.

| 조건 | 기준 | 제외 비율 |
|------|------|-----------|
| 극저유동성 | 20일 평균 거래대금 < 1억원 | ~16% |
| 우선주 | 종목명에 '우', '우B' 등 포함 | ~4% |
| 거래부재 | 20일 중 거래량=0 ≥ 5일 | ~4% |
| 가격고정 | 5일 종가 변동 없음 | ~4% |
| 거래량고사 | 120일 평균 거래량 < 1,000주 | ~3% |
| 비정상변동 | 60일간 ±29% 변동 ≥ 3회 | ~3% |
| 저가주 | 종가 < 500원 | ~2% |
| 하한가반복 | 20일간 -29% 하락 ≥ 2회 | ~0.04% |

평균 제외율: 17~23% (시장 상태에 따라 변동)

## 8. 향후 진행 계획

### Phase 1: 모델 고도화 (우선순위 높음)

   - C등급 내 트리거별 세분화 (event+고E는 B1으로 승격 검토)

3. **등급 기준 미세 조정** — 변별력 극대화를 위한 임계값 최적화

### Phase 2: 매매 시뮬레이션
1. **가상 계좌 시뮬레이션** — 등급별 전략(포지션/목표/손절)으로 실행
   - 수수료 0.015%, 세금 0.3%, 슬리피지 0.1% 반영
   - 일별 포트폴리오 구성, 동시 보유 종목 수 제한
2. **시장 환경 분기** — KOSPI가 60MA 위/아래에 따라 전략 자동 전환
   - 하락장: 목표수익 낮추고 빠른 익절
   - 상승장: 목표수익 높이고 스윙 홀딩

### Phase 3: 실시간 시스템
1. **실시간 포착 → 자동 스코어링** — 키움 API 또는 일봉 갱신 트리거
2. **알림 파이프라인** — 텔레그램/슬랙으로 A1/A2 종목 즉시 알림
3. **대시보드** — 일별 포착 종목, 등급 분포, 보유 종목 수익률 현황

### Phase 4: 고급 모델
1. **머신러닝 모델** — Random Forest / XGBoost로 등급 예측 고도화
2. **피처 엔지니어링** — BW 압축 기간, 이격 벡터, 거래량 에너지 등
3. **앙상블** — 규칙 기반 v2.0 + ML 모델 합산 점수
