# 키움 조건식 성과검증 & 폭발주 예측 스코어링 시스템

키움증권 HTS 조건식(60/200일 이동평균 돌파)이 포착하는 종목의 성과를 2년간 백테스트로 검증하고,
2축 스코어링 모델(안정성 S + 폭발력 E)로 고수익 종목을 사전에 예측하는 시스템입니다.

## 2축 스코어링 모델 v2.0

S-Score (안정성, 0-100) — 1개월 실현 수익 예측

- F1. 트리거 경로 (25점) — D+E 동시돌파가 최고
- F2. 60/200 이격률 (20점) — 3% 미만 극한수렴이 최적
- F3. RSI (15점) — 40-60 중립대가 최적
- F4. 볼린저밴드 폭 (15점) — 10-20 압축이 최적
- F5. 거래량 비율 (15점) — 5-10배가 최적
- F6. MA 기울기 (10점) — 양쪽 상승이 최적

E-Score (폭발력, 0-100) — 최고 수익 폭발력 예측

- G1. 종가 수준 (30점) — 3천원 미만 초저가가 최고
- G2. 당일 등락률 (25점) — 15%+ 급등이 최고
- G3. 거래량 비율 (20점) — 10배+ 극폭증이 최고
- G4. 볼린저밴드 폭 (15점) — 40+ 과확장이 최고
- G5. 트리거 경로 (10점) — event가 최고

## 등급 매트릭스

|  | E >= 60 (폭발력) | E < 60 (안정) |
|--|-----------------|--------------|
| S >= 70 | A1 (최우선) | A2 (안정우선) |
| 50 <= S < 70 | B1 (폭발후보) | B2 (표준) |
| S < 50 | C (관망) | D (패스) |

## 프로젝트 구조

| 파일 | 역할 |
|------|------|
| config.py | DB/HTS/백테스트 설정 |

| run_scan_backtest.py | 백테스트 실행 |
| scoring_model.py | 스코어링 모델 v2.0 |
| validate_scoring.py | 스코어링 역산 검증 |
| backtest_report.py | 결과 리포트 생성 |
| PROJECT_REPORT.md | 프로젝트 종합 보고서 |
| SESSION_HANDOFF.md | 세션 인수인계 문서 |

## 실행 방법

1. pip install -r requirements.txt
2. .env 파일에 DB 접속정보 입력
3. python precompute_indicators.py --start 2024-05-01 --end 2026-05-16
4. python run_scan_backtest.py --start 2024-05-19 --end 2026-05-16 --fast
5. python scoring_model.py
6. python validate_scoring.py
